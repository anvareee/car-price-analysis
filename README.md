# car-price-analysis
car price project using python

## Overview
This project analyzes a car price dataset using Python to identify key factors influencing car prices.

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## Key Steps
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Visualization of relationships between variables
- Identification of key price drivers

## Key Insights
- Mileage and brand significantly affect car prices
- Certain features show strong correlation with pricing trends

## Files
- Car_price_analysis.ipynb (main notebook)
- Car_price.csv (dataset)

## How to Run
1. Clone the repository
2. Open the notebook in Jupyter Notebook or VS Code
3. Run the cells sequentially
